# MegaConvert Glossary

Use these terms consistently across locales.

| Source term | Preferred meaning |
| --- | --- |
| Tool | Converter tool |
| Convert | Convert file |
| Plan | Subscription tier |
| Billing | Payments and plan management |
| Trial | Trial period |
| Promo code | Promo code |
| Account | User account |
| Connected accounts | Linked sign-in providers |
| Security | Session and access security |
| Active benefits | Active entitlements from promos |
| Promo history | Promo redemption history |
| Session | Signed-in device session |
| Logout all | Revoke all active sessions |

## Notes
- `Promo code` is a product concept, keep it short and recognizable.
- `Billing` is a section title, keep it concise.
- `Plan` refers to user access level, not technical plan.
