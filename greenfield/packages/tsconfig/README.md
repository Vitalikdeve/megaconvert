# @megaconvert/tsconfig

Shared TypeScript baselines for every workspace package and app.

- `base.json` - strict baseline shared everywhere.
- `nest-app.json` - Node and NestJS application settings.
- `next-app.json` - Next.js app settings.
- `node-library.json` - shared Node library settings.
- `react-library.json` - shared React library settings.
