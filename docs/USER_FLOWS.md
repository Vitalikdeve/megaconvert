# MegaConvert User Flows

## Flow 1: Basic Conversion

`Landing -> Upload -> AI analysis -> Choose action -> Processing -> Result -> Download`

## Flow 2: AI-assisted Conversion

`Upload -> AI recommendations -> Apply suggestion -> Auto-configured job -> Result summary`

## Flow 3: Upgrade to PRO

`Limit reached -> Upgrade prompt -> Plan selection -> Payment -> Subscription active`

## Flow 4: Generate API Key

`Account -> API tab -> Generate key -> Save securely`

## Flow 5: Share Result

`Job completed -> Click share -> Token generated -> Public link`
